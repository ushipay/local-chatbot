# depndency review actions
```
dependency-review-actionsでは以下のoutputsを提供している。
┌─────────────────────────┬──────────────────────────────────┐
  │         Output          │               内容               │
  ├─────────────────────────┼──────────────────────────────────┤
  │ comment-content         │ レビュー結果のレポートテキスト   │
  ├─────────────────────────┼──────────────────────────────────┤
  │ dependency-changes      │ 全依存関係の変更（JSON）         │
  ├─────────────────────────┼──────────────────────────────────┤
  │ vulnerable-changes      │ 脆弱性のある変更のみ（JSON）     │
  ├─────────────────────────┼──────────────────────────────────┤
  │ invalid-license-changes │ ライセンス違反の変更のみ（JSON） │
  ├─────────────────────────┼──────────────────────────────────┤
  │ denied-changes          │ 拒否ルールに該当する変更（JSON） │
  └─────────────────────────┴──────────────────────────────────┘
  また、ライセンスのホワイトリストは allow-licenses で設定できる
```
